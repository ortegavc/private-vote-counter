# Private Vote Counter
Alternate vote counting application for mayoral elections 2019 in the city of Machala, Ecuador.

## Tech

The application uses a number of open source projects:

- PHP
- MariaDB
- JavaScript
- jQuery
- Twitter Bootstrap- 
- CanvaJS
